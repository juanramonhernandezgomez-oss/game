# Colores al Centro

Mini juego multijugador local (2 jugadores) pensado para ser fácil de aprender y divertido para todas las edades.

## Cómo jugar

1. Abre `index.html` en un navegador móvil o escritorio.
2. Elige un modo (`Normal`, `Niños`, `Abuelos`).
3. Pulsa **Iniciar partida**.
4. Toca los cuadros del **color objetivo** para sumar puntos.
5. Si fallas, en modo Normal/Abuelos pierdes 1 punto y te congelas 1 segundo.
6. Gana quien tenga más puntos al terminar los 45 segundos.

## Características MVP

- Partidas rápidas de 45 segundos.
- Marcador para 2 jugadores en tiempo real.
- Cambio automático de color objetivo durante la ronda.
- Dificultad por modo para adaptar experiencia entre niños y adultos mayores.
